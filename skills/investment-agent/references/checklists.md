# Investment Checklists

## Pre-buy checklist

- Understand the asset in one plain sentence
- Know the main return driver
- Identify the 2-3 biggest risks
- Check valuation / pricing context if relevant
- Confirm time horizon matches the thesis
- Confirm position size would not cause forced selling
- Define what would invalidate the thesis
- Define what data or events to monitor after entry

## Watchlist template

For each asset, track:

- Ticker / name
- Why it is interesting now
- Desired entry logic
- Red flags
- Next catalyst
- Metrics to monitor
- Review frequency

## Ongoing monitoring

### Stocks
- Revenue growth
- Margin trend
- Cash flow
- Guidance changes
- Competitive position
- Valuation multiple vs history / peers

### ETFs / funds
- Expense ratio
- Index methodology
- Concentration
- Holdings drift
- Tracking quality
- AUM / liquidity

### Crypto
- Network / protocol usage
- Token issuance / unlocks
- Security / governance issues
- Regulatory developments
- Liquidity and custody considerations

# SAMPLE WORKFLOWS

## 1) Content creation workflow
1. Audience Researcher หา pain point / FAQ
2. Content Agent สร้าง content pack
3. Compliance check
4. Publish
5. บันทึกลง content calendar

## 2) Lead capture workflow
1. คนเห็น content แล้วทัก
2. CRM Agent สรุป intent
3. บันทึก lead tracker
4. ตั้ง stage และ due date

## 3) Conversation triage workflow
1. รับข้อความเข้า
2. classify product / business / unclear
3. ถามเพิ่ม 1–3 ข้อ
4. route ไป agent ที่เหมาะ

## 4) Follow-up workflow
1. เปิด due list
2. ส่ง follow-up ตามบริบท
3. บันทึกผล
4. ตั้งวันถัดไปหรือปิดเคส

## 5) Product recommendation routing workflow
1. CRM tag = product
2. Product Routing Agent จัดกรอบ need
3. เจ้าของธุรกิจตรวจและส่งข้อความ
4. บันทึกผลลัพธ์

## 6) Business-opportunity conversation routing workflow
1. CRM tag = business
2. Business Opportunity Agent คัดระดับความพร้อม
3. ส่งข้อมูลภาพรวมที่เหมาะ
4. นัดคุย human-led หากพร้อม

## 7) Performance review workflow
1. รวบรวม weekly KPI
2. Analyst สรุป funnel
3. หา 3 จุดที่ควรปรับ
4. Ops Manager สร้างแผนสัปดาห์ใหม่

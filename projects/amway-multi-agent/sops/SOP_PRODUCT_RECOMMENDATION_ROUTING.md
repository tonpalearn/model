# SOP_PRODUCT_RECOMMENDATION_ROUTING

## เป้าหมาย
ช่วย route บทสนทนาไปยังกรอบการแนะนำที่เหมาะ โดยไม่วินิจฉัยโรคและไม่ overclaim

## ขั้นตอน
1. ระบุ goal หลักของลูกค้า
2. ถามบริบทที่จำเป็นขั้นต่ำ
3. สรุป need เป็นภาษากลาง
4. เลือก asset / ข้อมูลที่เหมาะส่งต่อ
5. ให้เจ้าของธุรกิจตรวจถ้าประเด็นละเอียด

## คำถามขั้นต่ำ
- ตอนนี้โฟกัสเรื่องไหนมากที่สุดคะ
- มีข้อจำกัดหรือสิ่งที่กังวลเป็นพิเศษไหมคะ
- ก่อนหน้านี้เคยลองดูแลตัวเองแบบไหนมาบ้าง

## ห้าม
- ตอบเหมือนเป็นแพทย์
- ฟันธงผลลัพธ์
- เสนอสินค้าแบบไม่รู้บริบท

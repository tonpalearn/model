## B) Functional map: text vs video

## B.1 Text / image posts — สิ่งที่น่าจะทำหน้าที่
เมื่อเทียบกับคำอธิบายหน้าเพจที่พูดถึง “วิเคราะห์กราฟทองทุกวัน” และ “ห้อง Signal” โพสต์ข้อความหรือภาพนิ่งของ MTA น่าจะมีบทบาทดังนี้:

### บทบาทที่น่าจะเป็นไปได้สูง
- สรุปมุมมองตลาดรายวัน
- วาง key level หรือ bias ของทอง
- แจ้งมุมมองก่อนเข้า signal
- สรุปบทเรียนสั้น ๆ เรื่องการอ่านกราฟ
- โปรโมตห้อง signal / community / academy

### ภาษาที่ text post น่าจะใช้บ่อย
- buy / sell / wait
- แนวรับ / แนวต้าน / โซน
- กรอบราคา / จุดเข้า / เป้าหมาย / stop
- มุมมองวันนี้ / แผนวันนี้ / กราฟทองวันนี้
- signal / setup / confirmation

### สิ่งที่ text น่าจะเก่งกว่า video
- บีบข้อมูลให้สั้นและใช้งานได้เร็ว
- สื่อ bias และระดับราคาแบบตรงไปตรงมา
- ใช้เป็นโพสต์ประกาศหรือ reminder ได้ดี

---

## B.2 Video / reels — สิ่งที่น่าจะทำหน้าที่
เพราะ canonical reels page เปิด metadata ได้ชัด จึงมีน้ำหนักค่อนข้างมากว่า MTA ใช้วิดีโอสั้นเป็นช่องทางสำคัญ

### บทบาทที่น่าจะเป็นไปได้สูง
- เดินกราฟให้ดูเร็ว ๆ ว่าราคาเคลื่อนอย่างไร
- อธิบาย setup ของทองผ่านภาพมากกว่าข้อความ
- ดึง attention ด้วย hook สั้น
- ย้ำผลลัพธ์หรือจังหวะสำคัญของวัน
- พาคนจากคอนเทนต์ฟรีไปสู่ห้อง signal/community

### ภาษาที่ video/reel น่าจะใช้บ่อย
- ดูจุดนี้
- ทองวันนี้
- เข้าเมื่อยืนยัน
- หลุดโซน / ทดสอบโซน / กลับตัว
- เป้าต่อไป / แผนต่อไป / รอสัญญาณ

### สิ่งที่ reels น่าจะเก่งกว่า text
- ทำให้เห็น sequence ของราคา
- อธิบาย fake move / breakout / rejection ได้ชัดกว่า
- สร้างภาพจำของแบรนด์และ style การสอน
- เหมาะกับการสรุป daily gold move แบบรวดเร็ว

- before/after ของราคา
- จังหวะเข้าออกที่มองเห็นบนกราฟ

## E) Text-to-video relationship hypothesis

โมเดลที่น่าจะเป็นไปได้ที่สุดตอนนี้คือ:

### Text = “สรุป / บอกแผน / ชี้ระดับ”
- ใช้อ่านไว
- ใช้อ้างอิงก่อนเทรด
- ใช้เป็น call-to-action เข้าห้อง signal

### Video = “สาธิต / ทำให้เห็นภาพ / เร่งความเข้าใจ”
- ใช้ทำให้ pattern ติดตา
- ใช้สรุปการเคลื่อนไหวของทอง
- ใช้สร้างความน่าเชื่อถือและความรู้สึกว่าเพจ active จริงทุกวัน

ถ้าโมเดลนี้ถูกต้อง จุดแข็งของ MTA จะอยู่ที่การผสาน
- ข้อความที่ actionable
- กับวิดีโอที่ทำให้ setup เข้าใจง่ายขึ้น

---

## F) What remains unknown

ยังยืนยันไม่ได้ว่า:
- text post ลงรายละเอียดมากน้อยแค่ไหน
- video มี narration เชิงลึกหรือแค่คลิปสั้นโปรโมต
- สัดส่วนจริงระหว่าง educational content กับ promotional content เป็นเท่าไร
- reels ของเพจสอน setup เป็นระบบ หรือเน้นสรุปตลาด/โชว์ผลเป็นหลัก

# สรุปสั้น

## สิ่งที่เห็นจริง
- หน้าเพจประกาศชัดว่า **สอนเทรด Forex | วิเคราะห์กราฟทองทุกวัน | ห้อง Signal**
- หน้า **Reels** ของเพจมีอยู่จริงและเปิด metadata ได้

## สิ่งที่น่าจะจริง
- Text น่าจะใช้เพื่อสรุป bias, level, และ signal framing
- Video/Reels น่าจะใช้เพื่อ chart walkthrough, visual teaching, และการดึงความสนใจ
- ทองคำเป็นหัวข้อที่น่าจะโผล่บ่อยมากกว่าสินทรัพย์อื่น อย่างน้อยในระดับ positioning หน้าเพจ

## สิ่งที่ยังไม่รู้
- caption เต็มของโพสต์/รีล
- รายชื่อคลิปจริงรายชิ้น
- framework setup เฉพาะของ MTA

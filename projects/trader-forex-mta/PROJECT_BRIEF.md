# PROJECT BRIEF

ผู้ใช้ต้องการ:
- โปรเจกต์ Trader Forex
- ใช้หลาย agent ทำงานร่วมกัน
- ปลายทางคือ bot cAlgo / cTrader ที่รันได้จริง
- workflow ที่ต้องการ:
  1. Agent แกะวิธีคิดจาก Facebook Page MTA
  2. Agent นักวิเคราะห์ Price Action สร้าง pattern/logic
  3. Agent Dev เขียน cBot
  4. Agent Tester ทดสอบ logic และผลลัพธ์
  5. Agent PM คุมโครงการอัตโนมัติ

ข้อกำกับด้านความปลอดภัย:
- ห้ามอ้างว่าทำกำไรได้แน่นอน
- ต้องรายงานข้อจำกัดและความเสี่ยง
- ต้องทำให้ user รันเองได้ พร้อมเอกสาร

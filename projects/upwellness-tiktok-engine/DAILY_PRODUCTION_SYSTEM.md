# DAILY PRODUCTION SYSTEM

## เป้าหมายหลัก
ผลิตคอนเทนต์ให้ได้สม่ำเสมอ โดยไม่ทำให้คุณภาพตกหรือหลุด compliance

## Daily Output Modes

### Mode A: Sustainable (แนะนำเริ่มต้น)
- 2 ชิ้น/วัน
  - 1 video-style post
  - 1 static/slideshow-style post
- เวลารวมต่อวัน: 60-120 นาที
- เหมาะกับการรักษาความสม่ำเสมอ

### Mode B: Higher Output
- 4 ชิ้น/วัน
  - 2 video-style
  - 1 slideshow/static
  - 1 quick hook / reply / comment-bait post
- เวลารวมต่อวัน: 2.5-4 ชั่วโมง
- เหมาะเมื่อมีทีม หรือทำ batch ได้

## Daily Sequence
1. **Research/Trend scan** - หา 3-5 มุมประเด็น
2. **Select winners** - เลือก 2 ชิ้นสำหรับวันนี้
3. **Script/Caption drafting** - ทำสคริปต์ + caption + CTA
4. **Format split** - แยกเป็น video / static
5. **Compliance pass** - เช็กคำต้องห้ามและความเสี่ยง
6. **Publish prep** - เตรียม shot list / slide text / cover text / first comment
7. **Lead routing prep** - เตรียมคำตอบคอมเมนต์/DM
8. **Review next day** - ดูว่าอะไรน่าทำซ้ำหรือแตกไลน์

## Suggested Daily Cadence
- เช้า: วิดีโอ education หรือ energy hook
- เย็น: slideshow / save-worthy post / CTA post

## 7-Day Sustainable Plan
- Mon: Energy video + checklist slideshow
- Tue: Sleep hook video + short educational static
- Wed: Nutrition myth video + comment bait post
- Thu: Healthy aging talking-head + save post
- Fri: Story/opinion video + DM CTA post
- Sat: Routine video + carousel summary
- Sun: Weekly reflection / FAQ / community post

## Batch Workflow (ดีที่สุดสำหรับคนมีเวลาน้อย)
- วันจันทร์: คิดหัวข้อ 10-14 หัวข้อ
- วันอังคาร: เขียนสคริปต์ 6-8 ชิ้น
- วันพุธ: ถ่ายวิดีโอ batch
- วันพฤหัส: ทำ slideshow / covers / captions
- วันศุกร์: schedule / prepare replies / analyze

## Quality Gate ก่อนโพสต์
- Hook แข็งแรงใน 1 บรรทัดแรกหรือ 3 วินาทีแรกหรือไม่
- มีแค่ 1 message หลักหรือไม่
- ไม่มี disease claim หรือ result claim หรือไม่
- CTA ชัด แต่ไม่ aggressive หรือไม่
- ถ้าพูดถึงประสบการณ์ส่วนตัว แยกจากข้อเท็จจริงชัดหรือไม่

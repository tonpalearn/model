# HOOK LIBRARY

## Hook Style 1: เรียก pain point ตรงๆ
- ถ้าคุณตื่นมาแล้วเหมือนยังไม่ได้พัก คลิปนี้อาจเกี่ยวกับคุณ
- ถ้าบ่ายสองทีไรสมองตื้อทุกวัน ลองฟังอันนี้ก่อน
- บางคนไม่ได้ขี้เกียจ แต่ร่างกายกำลังส่งสัญญาณบางอย่าง

## Hook Style 2: ขัดความเชื่อเดิม
- ไม่ใช่ทุกอาการล้าจะแก้ได้ด้วยกาแฟ
- อาหารเช้าที่ดูเฮลท์ตี้ อาจทำให้พลังงานตกเร็วกว่าเดิม
- การนอนครบชั่วโมง ไม่ได้แปลว่าฟื้นตัวพอเสมอไป

## Hook Style 3: ตัวเลข / listicle
- 3 นิสัยที่ทำให้พลังงานตกแบบไม่รู้ตัว
- 5 สัญญาณว่าคุณอาจพักไม่พอ แม้จะนอนครบ
- 4 เรื่องเล็กๆ ที่คนอยากแก่ช้าควรเริ่มวันนี้

## Hook Style 4: curiosity gap
- มีสิ่งหนึ่งที่ทำให้หลายคนเหนื่อยทั้งวันโดยไม่รู้ตัว
- เหตุผลที่บางคนดูแลสุขภาพแล้วแต่ยังไม่สดชื่น อาจไม่ใช่เรื่องอาหารอย่างเดียว
- ถ้าคุณพยายามเฮลท์ตี้อยู่ แต่ยังรู้สึกไม่ดีขึ้น อาจข้ามจุดนี้ไป

## Hook Style 5: opinion / perspective
- เรื่องสุขภาพที่คนส่วนใหญ่มองข้าม ไม่ใช่วิตามิน แต่เป็นสิ่งนี้
- ถ้าจะเริ่มดูแลตัวเองใหม่ ฉันจะไม่เริ่มจากการหักดิบ
- ความจริงคือ คนส่วนใหญ่ไม่ได้ต้องการข้อมูลเพิ่ม แต่ต้องการระบบที่ทำตามได้

## Hook Style 6: comment bait
- อยากให้ทำพาร์ต 2 เรื่องนี้ไหม
- ถ้าคุณเป็นคนนอนแต่ไม่หายเพลีย คอมเมนต์คำว่า “เพลีย” เดี๋ยวทำ checklist ให้
- ถ้าอยากได้เวอร์ชันสำหรับคนทำงานออฟฟิศ คอมเมนต์ว่า “ออฟฟิศ”

## Hook Safety Notes
- หลีกเลี่ยงการพูดว่า “สัญญาณโรค” ถ้าไม่ได้อ้างอิงทางการแพทย์และมีบริบทชัด
- ใช้คำว่า “อาจ”, “บางคน”, “เป็นไปได้ว่า” เมื่อต้องพูดเชิงทั่วไป
- ห้ามใช้ hook ที่สื่อว่าหายโรค รักษาได้ หรือการันตีผล

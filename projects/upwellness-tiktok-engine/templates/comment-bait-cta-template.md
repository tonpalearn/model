# Comment Bait / CTA Template

## Safe CTA Patterns
- คอมเมนต์ว่า “เอา” ถ้าอยากได้ checklist
- คอมเมนต์ว่า “ต่อ” ถ้าอยากได้พาร์ต 2
- คอมเมนต์ว่า “นอน” ถ้าอยากได้ routine ก่อนนอน
- ทักคำว่า “เริ่ม” ถ้าอยากให้ช่วยดูว่าควรเริ่มตรงไหนก่อน

## Avoid
- DM ด่วน เดี๋ยวหายไม่ทัน
- รีบทักก่อนโปรหมด
- ถ้าไม่ทำตอนนี้จะสายเกินไป

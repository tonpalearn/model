# WORKFLOWS

## Workflow 1: Daily 2-Post Production
1. Trend/Research agent เสนอ 5 หัวข้อ
2. Ops/Editor agent เลือก 2 หัวข้อที่ balance ระหว่าง reach + trust + CTA
3. Script/Caption agent เขียน script/caption/hook/CTA
4. Video Concept agent ทำ shot structure สำหรับโพสต์วิดีโอ
5. Static/Image Post agent ทำ slide structure สำหรับโพสต์ภาพ
6. Compliance review pass
7. Ops/Editor agent finalize queue
8. Comment/DM lead-routing agent เตรียม reply pack

## Workflow 2: Batch Week Production
1. Research 20 ideas
2. Group by pillar
3. Select 14 pieces
4. Script 7 video + 7 static
5. Review compliance once per batch
6. Publish across week
7. Performance review agent สรุป lessons

## Workflow 3: Winning Post Expansion
ถ้าคอนเทนต์ตัวไหนเวิร์ก:
- แตกเป็นพาร์ต 2
- ทำซ้ำคนละ format
- ทำ FAQ reply post
- ทำ checklist download/DM bait version

## Workflow 4: Lead Routing
1. มีคนคอมเมนต์ keyword
2. ตอบ public แบบนุ่ม
3. เชิญ DM เมื่อเหมาะ
4. ถาม 2-3 qualifying questions
5. Route to education / product / business conversation
6. บันทึกใน lead tracker

## Workflow 5: Weekly Review
Performance review agent วิเคราะห์:
- hook ไหนหยุด scroll ได้ดี
- pillar ไหนได้ comment สูง
- CTA ไหนทำให้ DM start มากสุด
- หัวข้อไหนเสี่ยง compliance
- format ไหนคุ้ม effort มากสุด

จากนั้น Ops/Editor agent อัปเดตแผนสัปดาห์ถัดไป

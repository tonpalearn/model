# CTA AND LEAD ROUTING

## เป้าหมาย
เปลี่ยนความสนใจให้เป็นบทสนทนา โดยไม่กดดันและไม่ bait เกินจริง

## CTA Types

### 1) Low-friction Comment CTA
ใช้เมื่ออยากเพิ่ม engagement
- คอมเมนต์ว่า “เอา” ถ้าอยากได้ checklist
- คอมเมนต์ว่า “พาร์ต2” ถ้าอยากให้ขยายเรื่องนี้
- คอมเมนต์ว่า “นอน” ถ้าอยากได้ routine ก่อนนอน

### 2) Soft DM CTA
ใช้เมื่อเนื้อหาต้อง personalized
- ถ้าอยากให้ช่วยดูว่าเริ่มจากตรงไหนดี ทักมาได้
- ถ้าคุณอยากได้เวอร์ชันที่เหมาะกับชีวิตประจำวันของคุณ ส่งคำว่า “เริ่ม” มาได้

### 3) Qualification CTA
ใช้เมื่อเริ่มมี intent สูงขึ้น
- ตอนนี้คุณอยากโฟกัสเรื่องไหนมากกว่า: พลังงาน / การนอน / น้ำหนัก / วินัยสุขภาพ
- ถ้าสนใจทั้งด้านดูแลตัวเองและ community ที่ช่วยกันทำต่อเนื่อง ทักมาคุยได้แบบไม่ต้องรีบ

## Lead Routing Tree

### Stage 1: Public Comment
เป้าหมาย:
- ตอบแบบอบอุ่น
- ไม่ขายตรง
- เชิญเข้าบทสนทนาต่อเมื่อเหมาะ

ตัวอย่าง:
- "ได้เลย เดี๋ยวสรุปเวอร์ชันสั้นให้เพิ่มนะ"
- "ถ้าอยากได้แบบปรับให้เข้ากับชีวิตประจำวัน ทักมาได้ เดี๋ยวช่วยไล่ทีละข้อ"

### Stage 2: DM Entry
ถาม qualifying questions สั้นๆ:
1. ตอนนี้อยากโฟกัสเรื่องอะไรที่สุด
2. อะไรเป็นปัญหาหลักในชีวิตประจำวัน
3. เคยลองแก้อะไรมาบ้าง

### Stage 3: Route by Intent
- สนใจเนื้อหาความรู้ -> ส่ง resource / routine / checklist
- สนใจสินค้า -> พูดเชิงข้อมูลทั่วไป + แนะนำให้พิจารณาตามบริบทส่วนตัว
- สนใจโอกาสทางธุรกิจ -> คุยเรื่องเวลา เป้าหมาย และความคาดหวังอย่างตรงไปตรงมา โดยไม่การันตีรายได้

## Safe Language for Product/Business Bridge
- "ถ้าสนใจ ฉันเล่าแนวคิด/ตัวเลือกให้ฟังได้"
- "แล้วค่อยดูว่าเหมาะกับคุณไหม"
- "ไม่จำเป็นต้องรีบตัดสินใจ"
- "ผลลัพธ์ขึ้นกับแต่ละคนและวิธีลงมือทำ"

## Unsafe Language to Avoid
- "ตัวนี้ช่วยรักษา..."
- "รับรองเห็นผล"
- "ทำตามนี้แล้วหายแน่"
- "ทำธุรกิจนี้รายได้แน่นอน"
- "เข้ามาแล้วชีวิตเปลี่ยนทันที"

## Simple DM Flow
1. ขอบคุณที่ทัก
2. ถาม focus area
3. สรุปสิ่งที่เข้าใจ
4. ให้คำแนะนำเบื้องต้นหรือ resource
5. ถ้าเหมาะ ค่อย bridge ไป product/business conversation
6. ปิดด้วย open loop ไม่ pressure

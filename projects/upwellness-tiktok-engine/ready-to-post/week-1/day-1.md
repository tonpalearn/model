# DAY 1

## Post A — Talking-head Video
**Title/Hook:** ถ้าคุณนอนครบ แต่ยังตื่นมาเหมือนไม่ได้พัก ปัญหาอาจไม่ใช่แค่จำนวนชั่วโมง
**Format:** Talking-head / educational
**Content Goal:** สร้าง relatability + trust จาก pain point เรื่องหลับแต่ไม่สดชื่น
**Persona:** คนทำงานพลังงานตก / คนเริ่มใส่ใจสุขภาพวัย 35+
**Pillar:** Sleep & Recovery

**Script:**
ถ้าคุณนอนครบ แต่ตอนเช้ายังรู้สึกเหมือนแบตไม่เต็ม ปัญหาอาจไม่ใช่แค่คุณนอนกี่ชั่วโมง
หลายคนเช็กแค่เวลาเข้านอนกับเวลาตื่น แต่ไม่ได้เช็กว่าก่อนนอนเราเปิดทางให้ร่างกายฟื้นตัวจริงไหม
ถ้าก่อนนอนยังตอบงาน ไถมือถือ กินหนัก หรือปล่อยให้หัววิ่งไม่หยุด ร่างกายอาจยังไม่เข้าโหมดพักเต็มที่
ถ้าอยากเริ่มแบบง่าย ลอง 3 อย่างนี้ก่อน
หนึ่ง ลดแสงจากหน้าจอช่วง 30 นาทีสุดท้าย
สอง ปิดงานให้จบก่อนขึ้นเตียง
สาม อย่ารอให้เพลียจนหลับคาโทรศัพท์
มันไม่ใช่สูตรวิเศษนะ แต่เป็นจุดเริ่มที่หลายคนมองข้าม

**Caption:**
บางทีเราไม่ได้ขาดความพยายาม แต่อาจยังไม่มี routine ที่ช่วยให้ร่างกายรู้ว่า “ถึงเวลาพักแล้ว”

**CTA / Comment Bait:**
ถ้าอยากได้ routine ก่อนนอนแบบสั้นๆ สำหรับคนงานเยอะ คอมเมนต์ว่า “นอน”

**Visual Direction:**
- เปิดด้วยหน้าคนพูดครึ่งตัว + ข้อความบนจอ: “นอนครบ แต่ไม่สดชื่น?”
- ตอนพูด 3 ข้อ ให้ขึ้นเป็น text bullet สั้นๆ ทีละข้อ
- ปิดท้ายด้วยเฟรมค้าง 1 วิ: “อยากได้ routine ก่อนนอน คอมเมนต์: นอน”

## Post B — Slideshow / Static
**Title/Hook:** 3 เหตุผลที่คุณง่วงบ่าย ทั้งที่กาแฟก็แล้ว งานก็ต้องไปต่อ
**Format:** Slideshow / save-worthy
**Content Goal:** เพิ่ม save + share + comment จาก pain point ชัดมากในชีวิตประจำวัน
**Persona:** คนทำงานพลังงานตก
**Pillar:** Daily Energy

**Slide Copy:**
1. 3 เหตุผลที่คุณง่วงบ่าย ทั้งที่กาแฟก็แล้ว งานก็ต้องไปต่อ
2. มื้อแรกหวานหรือเบาเกินไป พลังงานเลยขึ้นไวแล้วตกไว
3. นั่งยาวทั้งวัน ร่างกายเหมือนค้างโหมด
4. ดื่มน้ำน้อยกว่าที่คิด ก็มีผลกับความสดชื่น
5. เริ่มง่ายๆ: เช็กมื้อแรก + ลุกขยับ 3-5 นาที + เช็กน้ำระหว่างวัน
6. ถ้าอยากได้เวอร์ชันสำหรับคนทำงานออฟฟิศ คอมเมนต์ว่า “ออฟฟิศ”

**Caption:**
ง่วงบ่ายไม่ได้แปลว่าคุณขี้เกียจเสมอไป บางทีเป็นเรื่องพลังงานที่แกว่งทั้งวันมากกว่า

**CTA / Comment Bait:**
คอมเมนต์ว่า “ออฟฟิศ” ถ้าอยากได้เวอร์ชันจัดให้สำหรับคนประชุมทั้งวัน

**Visual Direction:**
- โทนสีสะอาด อ่านง่าย
- หน้าแรกใช้ประโยค hook ใหญ่ๆ
- สไลด์ 2-4 ใช้ไอคอนง่ายๆ: จานอาหาร / เก้าอี้ / แก้วน้ำ
- สไลด์สุดท้ายทำเป็น save prompt เล็กๆ: “เซฟไว้เช็กตอนบ่ายได้”

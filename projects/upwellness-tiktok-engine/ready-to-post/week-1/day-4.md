# DAY 4

## Post A — Talking-head Video
**Title/Hook:** ทำไมบางคนดูแลตัวเองแล้ว แต่ยังไม่รู้สึกดีขึ้น
**Format:** Talking-head / curiosity
**Content Goal:** เปิด curiosity gap และชวนคนรู้สึกว่า “ฉันไม่ได้พลาดคนเดียว”
**Persona:** คนทำงานพลังงานตก / biohack-curious / คนเริ่มใส่ใจสุขภาพวัย 35+
**Pillar:** Wellness Myths & Trend Decode

**Script:**
มีคนจำนวนมากที่พยายามดูแลตัวเองแล้ว แต่ยังรู้สึกว่าไม่ค่อยดีขึ้น
ไม่ใช่เพราะคุณไม่พยายามพอเสมอไป
บางทีคุณกำลังทุ่มแรงในจุดที่ไม่ใช่คานงัดหลักของชีวิตตอนนี้
เช่น ไปโฟกัสของเสริมก่อน แต่การนอนยังพัง ตารางชีวิตยังเหวี่ยง หรือความเครียดยังสูงจนร่างกายฟื้นตัวยาก
ฉันเลยชอบเริ่มจากคำถามง่ายๆ มากกว่า ว่าตอนนี้คอขวดจริงๆ คือการนอน พลังงาน อาหาร หรือความสม่ำเสมอ
ถ้าหาจุดนั้นเจอ การดูแลตัวเองจะไม่รู้สึกมั่วไปหมด

**Caption:**
บางทีปัญหาไม่ใช่คุณไม่มีวินัย แต่ยังไม่เจอ “คานงัดหลัก” ของตัวเอง

**CTA / Comment Bait:**
ถ้าอยากให้ฉันแตกต่อเรื่อง “หาจุดเริ่มต้นยังไง” คอมเมนต์ว่า “แตกต่อ”

**Visual Direction:**
- โทนพูดนิ่ง มั่นใจ แบบคนผ่านการลองผิดลองถูกมาแล้ว
- เปิดด้วย text: “ดูแลตัวเองแล้ว แต่ยังไม่ดีขึ้น?”
- ตัดคำว่า นอน / พลังงาน / อาหาร / ความสม่ำเสมอ ขึ้นจอทีละคำ

## Post B — Slideshow / Static
**Title/Hook:** 3 จุดเช็กเบื้องต้น ก่อนโทษตัวเองว่าไม่มีวินัย
**Format:** Slideshow / educational
**Content Goal:** แก้ self-blame และเพิ่มความรู้สึกว่า account นี้เข้าใจชีวิตจริง
**Persona:** คนทำงานพลังงานตก / คุณแม่หรือผู้หญิงดูแลหลายบทบาท
**Pillar:** Real-Life Routine & Personal Perspective

**Slide Copy:**
1. 3 จุดเช็กเบื้องต้น ก่อนโทษตัวเองว่าไม่มีวินัย
2. คุณพักพอจริงไหม หรือแค่พยายามฝืนต่อ
3. สิ่งที่ตั้งใจทำมันยากเกินชีวิตจริงตอนนี้ไหม
4. คุณมีระบบเตือน/เตรียมไว้ หรือหวังพึ่งแรงฮึดอย่างเดียว
5. วินัยที่ยั่งยืน มักเริ่มจากระบบที่เล็กพอจะทำซ้ำได้
6. ถ้าอยากได้ตัวอย่างระบบง่ายๆ คอมเมนต์ว่า “ระบบ”

**Caption:**
บางทีสิ่งที่ต้องแก้ก่อน ไม่ใช่นิสัยเสีย แต่เป็นระบบที่ไม่ช่วยให้ทำต่อได้

**CTA / Comment Bait:**
คอมเมนต์ว่า “ระบบ” ถ้าอยากได้ตัวอย่างวิธีจัดให้ทำตามง่ายขึ้น

**Visual Direction:**
- ใช้ดีไซน์เรียบ น่าอ่าน เหมือนโพสต์ที่คนอยากส่งให้เพื่อน
- เน้นสไลด์ 5 ด้วยตัวหนา: “เล็กพอจะทำซ้ำได้”

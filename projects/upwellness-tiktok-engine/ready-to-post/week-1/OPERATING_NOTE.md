# OPERATING NOTE

## วิธีใช้แบบเร็วที่สุด
1. โพสต์ `Post A` ตอนเช้าหรือช่วงกลางวัน
2. โพสต์ `Post B` ตอนเย็นหรือช่วงคนมีเวลานั่งอ่าน
3. ใช้ caption ตามไฟล์ได้เลย หรือปรับคำให้เข้ากับน้ำเสียงคนพูด
4. ถ้ามีคนคอมเมนต์ตาม CTA ให้ตอบสั้น อบอุ่น และชวนคุยต่อแบบไม่เร่ง
5. ถ้ามี intent สูงค่อยชวน DM ด้วยคำง่ายๆ เช่น
   - "ได้เลย ถ้าอยากได้เวอร์ชันที่เข้ากับชีวิตประจำวันคุณ ทักมาได้"
   - "ถ้าสะดวก ทักคำว่า เริ่ม มาได้ เดี๋ยวช่วยไล่ให้ว่าควรเริ่มจากตรงไหนก่อน"

## สูตรเลือกโพสต์แต่ละวัน
- อยากได้ reach / หยุด scroll -> ใช้ Post A
- อยากได้ save / share / comment -> ใช้ Post B
- ถ้าวันไหนเวลาไม่พอ ให้โพสต์แค่ 1 ชิ้น โดยเลือก Post A ก่อน

## กติกาก่อนโพสต์
- อย่าเติมคำแนวรักษาโรค / หาย / รับรองเห็นผล
- ถ้าพูดเรื่องสินค้า ให้คุยเชิง routine support เท่านั้น และพาไปคุยส่วนตัวเมื่อเป็นเรื่องเฉพาะบุคคล
- ถ้ามีคนถามอาการเฉพาะ ให้ตอบแบบข้อมูลทั่วไป และแนะนำปรึกษาผู้เชี่ยวชาญเมื่อเหมาะ

## คำตอบคอมเมนต์สั้นๆ ที่หยิบใช้ได้
- "ได้เลย เดี๋ยวทำเวอร์ชันสั้นให้ต่อในโพสต์ถัดไป"
- "อันนี้หลายคนเป็นเหมือนกันเลย ถ้าอยากได้เวอร์ชันปรับกับชีวิตประจำวัน ทักมาได้"
- "จริงมาก เรื่องนี้ไม่ใช่แค่แรงใจ แต่เป็นเรื่องระบบที่ทำตามได้"
- "ถ้าอยากให้แตกต่อเป็น checklist เดี๋ยวทำให้ได้"

## สิ่งที่ทำได้ทันที
- เปิดไฟล์ตามวัน แล้วอ่านตามสคริปต์ได้เลย
- หรือคัดลอก slide copy ไปใส่ Canva / CapCut ได้ทันที
- ใช้ CTA เดิมได้เลยโดยไม่ต้องคิดใหม่

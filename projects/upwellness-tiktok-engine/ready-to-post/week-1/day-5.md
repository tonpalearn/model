# DAY 5

## Post A — Talking-head Video
**Title/Hook:** ความจริงเรื่องพลังงานตกที่กาแฟอย่างเดียวช่วยไม่พอ
**Format:** Talking-head / educational
**Content Goal:** ขัดความเชื่อเดิม + เปิดทางสู่การดูแลตัวเองแบบองค์รวม
**Persona:** คนทำงานพลังงานตก
**Pillar:** Daily Energy

**Script:**
ฉันไม่ได้บอกว่ากาแฟไม่ดีนะ แต่ถ้าพลังงานคุณตกทุกวัน กาแฟอย่างเดียวอาจช่วยไม่พอ
เพราะสิ่งที่อยู่ใต้เรื่องนี้ บางทีกลับเป็นการนอนที่ไม่ฟื้นตัว มื้อแรกที่พยุงพลังงานไม่อยู่ หรือการนั่งยาวจนร่างกายเหมือนปิดเครื่องครึ่งหนึ่ง
กาแฟอาจช่วยให้ผ่านช่วงนั้นไปได้ แต่ไม่ได้แปลว่าต้นเหตุถูกจัดการแล้ว
ถ้าอยากให้พลังงานนิ่งขึ้นจริง ลองมองทั้งวัน ไม่ใช่มองแค่แก้วในมือ
เริ่มจากคำถามง่ายๆ วันนี้คุณนอนยังไง กินอะไรเป็นมื้อแรก แล้วได้ขยับตัวบ้างไหม

**Caption:**
กาแฟอาจเป็นตัวช่วยระยะสั้น แต่พลังงานระยะยาวมักมาจากระบบชีวิตทั้งวัน

**CTA / Comment Bait:**
ถ้าอยากได้ “3 จุดเช็กพลังงาน” แบบเร็วๆ คอมเมนต์ว่า “พลังงาน”

**Visual Direction:**
- เริ่มจากช็อตถือแก้วกาแฟแล้วมองกล้อง
- ตัด b-roll โต๊ะทำงาน / คนลุกเดิน / มื้อเช้าง่ายๆ
- ขึ้น text สั้น: “กาแฟ ≠ คำตอบทั้งหมด”

## Post B — Engagement / Comment Post
**Title/Hook:** ตอนนี้คุณอยากโฟกัสเรื่องไหนมากที่สุด: นอน / พลังงาน / อาหาร / วินัย
**Format:** Static engagement post
**Content Goal:** ดึงคอมเมนต์ + เก็บ intent ของคนดูเพื่อนำไปแตกคอนเทนต์และต่อบทสนทนา
**Persona:** ทุก persona โดยเฉพาะคนที่มี intent จะคุยต่อ
**Pillar:** Community & Soft Lead-In

**Post Copy:**
ตอนนี้ถ้าต้องเลือกแค่ 1 เรื่อง
คุณอยากโฟกัสอะไรมากที่สุด
- นอน
- พลังงาน
- อาหาร
- วินัยสุขภาพ

ไม่มีคำตอบถูกผิดนะ
แค่อยากรู้ว่าตอนนี้ชีวิตจริงของแต่ละคนติดตรงไหนมากที่สุด

**Caption:**
คอมเมนต์คำเดียวก็ได้ เดี๋ยวฉันเอาไปทำโพสต์ต่อให้ตรงที่สุด

**CTA / Comment Bait:**
พิมพ์แค่ 1 คำ: “นอน” / “พลังงาน” / “อาหาร” / “วินัย”

**Soft DM Bridge:**
ถ้าใครอยากคุยแบบส่วนตัวว่าเริ่มจากตรงไหนถึงจะเข้ากับชีวิตตัวเอง ทักคำว่า “เริ่ม” มาได้

**Visual Direction:**
- ใช้ดีไซน์เรียบมาก เหมือนโพลล์
- ให้ 4 คำใหญ่ ชัด อ่านจบไว
- ปักคอมเมนต์แรก: “ฉันจะเอาหัวข้อที่คนตอบเยอะสุดไปทำต่อ”

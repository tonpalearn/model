# Post 03: อาหารเช้าดูเฮลท์ตี้ แต่หิวเร็ว

- **Format recommendation:** 3-slide
- **Goal / persona:** ให้ข้อมูลที่เอาไปใช้ได้ทันทีและชวนคอมเมนต์ | คนทำงานพลังงานตก / คุณแม่เวลาน้อย
- **Visual style note:** ใช้รูปทรงบล็อก 3 ช่อง สะอาด พรีเมียม โทนขาว-เขียวอ่อน

## Final on-screen text copy (ready for Canva)
1. SLIDE 1
อาหารเช้าที่ดู healthy
อาจยังไม่ช่วยให้อิ่มนาน

2. SLIDE 2
ถ้ามีแต่หวานง่าย
หรือเบาเกินไป
พลังงานอาจขึ้นไวแล้วลงไว

3. SLIDE 3
ลองคิดมื้อเช้าแบบนี้
โปรตีน + ไฟเบอร์ + น้ำ
เรียบง่าย แต่อยู่ท้องกว่า

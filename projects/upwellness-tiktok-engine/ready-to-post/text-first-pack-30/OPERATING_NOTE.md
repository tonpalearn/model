# OPERATING NOTE

## เป้าหมายการใช้งาน
แพ็กนี้ออกแบบมาเพื่อให้ทีม UPWellness โพสต์ได้เร็ว โดยใช้ **Canva template system เดียว** แล้วเปลี่ยนแค่ข้อความและสี accent เล็กน้อย

## Single Canva Template System
แนะนำให้มี template หลักแค่ 3 ขนาดการใช้งาน:
1. **Text Card / One-screen**
2. **3-slide Carousel**
3. **5-slide Carousel**

แต่ทั้งหมดควรใช้ design language เดียวกัน:
- ฟอนต์ไทยหลัก 1-2 ฟอนต์
- พื้นหลังเรียบหรือ texture เบามาก
- accent color เดียวของแบรนด์
- spacing เยอะ อ่านง่าย
- 1 slide = 1 ความคิดหลัก

## Workflow 5 นาทีต่อโพสต์
1. เลือกโพสต์จาก `pack-index.csv`
2. เปิดไฟล์โพสต์ใน `posts/`
3. copy on-screen text ไปวางใน template
4. ปรับขนาดตัวอักษรให้ “อ่านจบไว”
5. ใส่ caption ลง TikTok
6. วาง CTA ให้ชัด แต่ไม่ hard sell
7. ตรวจ compliance ก่อนกดโพสต์

## Rules for Fast Execution
- อย่าเปลี่ยนคำเยอะ ถ้าต้นฉบับคมอยู่แล้ว
- ถ้าข้อความยาวเกินจอ ให้ตัดเป็น 3-slide แทน one-screen
- ถ้าโพสต์เป็นแนว opinion ให้คงน้ำเสียง “มุมมอง” ไม่ใช่ข้อสรุปทางการแพทย์
- ถ้าโพสต์ชวน DM ให้ใช้ถ้อยคำสุภาพ เช่น “ทักมาได้”, “ค่อยคุยกันได้”

## Recommended Posting Mix
สำหรับแพ็กนี้ แนะนำใช้แบบนี้:
- วันธรรมดา: เช้า 1 / เย็น 1
- เช้า = pain point, energy, sleep, mindset
- เย็น = checklist, soft routine, comment bait, soft lead-in

## Caption Style
caption ควรเป็น:
- 2-4 ประโยค
- อ่านง่าย ไม่วิชาการ
- practical
- ปิดด้วย CTA เดียว

## Compliance Guardrails ใช้งานจริง
ก่อนโพสต์ทุกครั้ง เช็ก 5 ข้อนี้:
1. มีคำเหมือนรักษาโรคไหม
2. มีคำเหมือนการันตีผลไหม
3. มีการเหมารวมว่าใช้ได้กับทุกคนไหม
4. ถ้าพูดถึง trend / support มีบริบทพอไหม
5. CTA ดูสุภาพ ไม่กดดันไหม

## Soft Bridge ไปบทสนทนา
ถ้าโพสต์ไหนมี intent สูง สามารถตอบคอมเมนต์หรือ DM ด้วยรูปแบบนี้:
- ขอบคุณที่คอมเมนต์
- ถามว่าเขาอยากโฟกัสเรื่องอะไรที่สุด
- ให้แนวทางพื้นฐานก่อน
- ถ้าเหมาะ ค่อย bridge ไปคุยเรื่อง routine, support, products หรือ community

## Best Practice
- ทำซีรีส์จากโพสต์ที่คนคอมเมนต์เยอะ
- โพสต์ที่มีเลข 1/2/3/4 มักช่วยให้คนตอบง่าย
- โพสต์ one-screen เหมาะกับ hook แรงๆ
- โพสต์ 3-slide เหมาะกับ insight + reframe
- โพสต์ 5-slide เหมาะกับ listicle / checklist / step-by-step

# CANVA LAYOUT FORMULAS

ใช้สูตรเดิมวนได้ทั้งแพ็กนี้ เพื่อให้ผลิตเร็วและภาพรวมช่องสม่ำเสมอ

## Formula 1: Bold Hook Card
เหมาะกับ: one-screen text

โครง:
- บรรทัด 1 = hook ใหญ่สุด
- บรรทัด 2-4 = supporting line
- บรรทัดสุดท้าย = punch line หรือ reframe

ทริก:
- เน้น 1 วลีด้วย accent color
- ไม่เกิน 5-6 บรรทัด
- มี margin เยอะ

## Formula 2: Pain → Reframe → Action
เหมาะกับ: 3-slide

โครง:
- Slide 1 = pain / hook
- Slide 2 = reframe / insight
- Slide 3 = action / takeaway / CTA

ทริก:
- สไลด์ 1 ต้องอ่านจบไวมาก
- สไลด์ 2 ให้ความรู้สึก “อ๋อ”
- สไลด์ 3 ปิดแบบ practical

## Formula 3: Numbered Micro-List
เหมาะกับ: 5-slide

โครง:
- Slide 1 = headline + number
- Slide 2-4 = list item
- Slide 5 = takeaway or CTA

ทริก:
- ใช้เลขใหญ่ชัด
- 1 slide = 1 item
- อย่าใส่เกิน 10-14 คำต่อบล็อกหลัก

## Formula 4: Quote + Reality Line
เหมาะกับ: one-screen text หรือ slide เปิด

โครง:
- ประโยคแรก = ความเชื่อ / สิ่งที่คนพูดกันบ่อย
- ประโยคสอง = reality line หรือ reframe
- ปิดด้วยประโยคคมๆ

ทริก:
- เหมาะกับโพสต์ opinion / mindset / premium tone
- ใช้พื้นที่ว่างเยอะ จะดูแพงกว่าเยอะ

## Formula 5: Simple Checklist Board
เหมาะกับ: 3-slide หรือ 5-slide

โครง:
- Slide 1 = context
- Slide 2-4 = checklist / steps
- สไลด์สุดท้าย = “เริ่มจากแค่นี้พอ” หรือ CTA

ทริก:
- ใช้ bullet หรือเลขสั้นๆ
- เหมาะกับโพสต์ save-worthy
- ถ้าจะใช้ไอคอน ใช้แค่ 1 style ตลอดทั้ง pack

## Design Direction รวม
- Thai-first
- premium minimal
- clean spacing
- ฟอนต์อ่านง่ายบนมือถือ
- ไม่ใช้ลูกเล่นเยอะ
- ให้คนอ่านจบภายใน 1-2 วินาทีต่อ slide

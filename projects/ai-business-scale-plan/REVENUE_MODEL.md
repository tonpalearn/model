# REVENUE MODEL

## 1) เป้ารายได้
เป้าหมาย 10,000,000 บาทต่อปี
เฉลี่ยประมาณ 833,333 บาทต่อเดือน

## 2) Reality check
ถ้าต้นขายเวลาแบบเดิมอย่างเดียว เช่น
- online 1:1 = 4,500 บาท / 4 ชั่วโมง
- offline 1:1 = 7,000 บาท / 4 ชั่วโมง
- งานกลุ่มบางวัน = 10,000 ถึง 12,000 บาท

โมเดลนี้ยัง scale ยาก เพราะ
- ชั่วโมงต่อวันมีจำกัด
- ต้องใช้ต้นลงมือเองเกือบทั้งหมด
- รายได้ไม่ predictable มากพอ

## 3) Revenue mix ที่แนะนำ
### Model A: Balanced mix
- B2C 1:1 / small group = 100,000 / เดือน
- Corporate training = 400,000 / เดือน
- Consulting / implementation = 250,000 / เดือน
- Digital products / membership = 80,000 / เดือน

รวม = 830,000 / เดือน

## 4) Pricing ladder ที่แนะนำ
### B2C
- 1:1 online 2 ชั่วโมง = 3,500 ถึง 5,000
- 1:1 intensive 4 ชั่วโมง = 6,000 ถึง 9,000
- small group workshop = 1,490 ถึง 3,990 ต่อคน
- cohort bootcamp = 4,900 ถึง 15,000 ต่อคน

### B2B Training
- mini workshop 2 ชั่วโมง = 15,000 ถึง 30,000
- half-day workshop = 30,000 ถึง 60,000
- full-day workshop = 60,000 ถึง 120,000
- customized corporate bootcamp = 80,000 ถึง 200,000+

### Consulting / Implementation
- AI audit / roadmap = 25,000 ถึง 60,000
- 30-day AI adoption sprint = 60,000 ถึง 150,000
- internal prompt / workflow system setup = 80,000 ถึง 250,000
- monthly retainer = 25,000 ถึง 100,000+

### Digital product
- prompt pack / template = 490 ถึง 1,990
- recorded course = 1,990 ถึง 7,900
- membership / community = 490 ถึง 1,500 ต่อเดือน

## 5) ตัวอย่างวิธีไปถึง 10 ล้าน
### Scenario 1: Corporate-led
- corporate workshop เฉลี่ย 70,000 บาท × 6 งาน / เดือน = 420,000
- consulting package เฉลี่ย 90,000 บาท × 2 งาน / เดือน = 180,000
- B2C / group class = 120,000
- digital products = 80,000

รวม = 800,000 / เดือน
รวมต่อปี ≈ 9.6 ล้านบาท

ถ้าเพิ่ม upsell อีกเล็กน้อยหรือปิดงานใหญ่เพิ่ม 1-2 งานต่อไตรมาส จะทะลุ 10 ล้านได้

### Scenario 2: Mixed engine
- workshop 4 งาน / เดือน × 60,000 = 240,000
- implementation 2 งาน / เดือน × 120,000 = 240,000
- bootcamp / cohort 30 คน / เดือน × 4,000 = 120,000
- 1:1 + small group = 120,000
- digital product / membership = 100,000

รวม = 820,000 / เดือน

## 6) ข้อเสนอเชิงกลยุทธ์เรื่องราคา
ต้นควรขึ้นราคาบางส่วน ไม่ควรติดกับราคาเดิมนานเกินไป
เพราะตอนนี้ต้นไม่ได้ขายแค่เวลา แต่ขาย
- know-how
- speed
- clarity
- business result

หลักการ
- อย่าขายถูกเพื่อเอาง่าย
- ทำ package ให้ชัด
- ขายผลลัพธ์แทนชั่วโมง
- มี 3 tier ให้ลูกค้าเลือก

## 7) Conversion ladder
ลูกค้าไหลแบบนี้
- content / social / Fastwork / referral
-> mini class / free session / intro call
-> paid workshop / 1:1
-> consulting package
-> implementation
-> retainer / repeat training

## 8) KPI ที่ควรวัด
- leads ต่อเดือน
- intro calls ต่อเดือน
- conversion rate เป็น paid client
- average ticket size
- revenue by segment
- repeat client rate
- referral rate
- utilization time ของต้น

## 9) ข้อสรุป
เป้าหมาย 10 ล้านไม่ใช่เรื่องของ “รับงานเยอะขึ้นอย่างเดียว”
แต่เป็นเรื่องของ
- ราคาเฉลี่ยสูงขึ้น
- งานองค์กรเยอะขึ้น
- มีของขายที่ทำซ้ำได้
- upsell ต่อจากการสอนเป็น consult / implementation

# Resume Project — Head of AI School

ไฟล์ชุดนี้ใช้สำหรับประเมินความเหมาะสมและปรับ narrative ของ Chaiwat Phuwanonjirakawin ให้สมัครบทบาทแนว Head of AI School / AI education leadership ได้ดีขึ้น

## Files
- `FIT_ASSESSMENT.md` — วิเคราะห์ว่า match แค่ไหน จุดแข็ง/จุดเสี่ยงคืออะไร
- `TAILORED_RESUME.md` — resume เวอร์ชันรีเฟรมให้เข้ากับบทบาท
- `COVER_LETTER.md` — ร่างจดหมายสมัครงาน
- `INTERVIEW_POSITIONING.md` — แนวตอบคำถามสำคัญในการสัมภาษณ์

## Recommended use
1. อ่าน `FIT_ASSESSMENT.md` ก่อน
2. ใช้ `TAILORED_RESUME.md` เป็นฐานไปจัด format จริงใน CV
3. ใช้ `COVER_LETTER.md` ปรับ tone ตามบริษัท/ผู้รับ
4. ใช้ `INTERVIEW_POSITIONING.md` เตรียมตอบสัมภาษณ์

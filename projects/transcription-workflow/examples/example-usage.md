# Example Usage

## Voice note

```bash
bash scripts/transcribe_local.sh /Users/ckawin/.openclaw/media/inbound/file.ogg medium
```

## Save result

```bash
bash scripts/transcribe_local.sh /Users/ckawin/.openclaw/media/inbound/file.ogg medium > note.txt
```

## Then ask the assistant

- turn this into Thai meeting notes
- clean this into readable transcript
- summarize key action items
- rewrite this into a post draft

# Automation Map - ต้นทาง AI

## เป้าหมาย automation
- เก็บรายชื่อคนที่คอมเมนต์หรือทักมา
- ส่ง lead magnet ให้อัตโนมัติหรือกึ่งอัตโนมัติ
- คัดแยกกลุ่มลูกค้า
- follow-up เพื่อปิดการขาย Prompt Pack
- ส่งต่อคนที่สนใจลึกไปยังบริการ audit/advisory

## Flow หลัก
โพสต์ Facebook -> คนคอมเมนต์ keyword -> ระบบ/คนส่ง DM -> ส่งไฟล์ฟรี -> เก็บข้อมูลลงชีต -> ถามคัดกรอง -> เสนอ Prompt Pack -> follow-up -> นัดคุยบริการ

## เวอร์ชัน 1: Manual + Semi-auto
เหมาะเริ่มต้นที่สุด

เครื่องมือ:
- Facebook Page inbox
- Google Drive (เก็บไฟล์)
- Google Sheets (เก็บ lead)
- Template ข้อความตอบ

ขั้นตอน:
1. โพสต์พร้อม CTA เช่น พิมพ์ PROMPT
2. เช็กคอมเมนต์วันละ 2–3 รอบ
3. ส่ง DM ด้วยข้อความ template
4. วางลิงก์ไฟล์ฟรีจาก Google Drive
5. บันทึกชื่อ/เพจ/ความสนใจ ลง Google Sheets
6. ถามต่อว่าทำธุรกิจอะไร
7. ถ้าเข้ากลุ่มเป้าหมาย เสนอ Prompt Pack
8. ถ้าสนใจลึก เสนอ AI Starter Audit

ข้อดี:
- เริ่มได้เลย ไม่ต้องต่อ API
- คุมคุณภาพข้อความได้

ข้อเสีย:
- ใช้แรงคน
- โตแล้วจะเหนื่อย

## เวอร์ชัน 2: Low-code Automation
เมื่อเริ่มมี lead มากขึ้น

เครื่องมือแนะนำ:
- Make.com หรือ Zapier
- Google Sheets / Airtable
- Gmail / LINE OA / webhook (ถ้ามี)
- ระบบจ่ายเงิน + ฟอร์มรับออเดอร์

Flow ตัวอย่าง:
1. มีคนกรอกฟอร์มรับ freebie หรือสั่งซื้อ
2. ข้อมูลเข้า Google Sheets/Airtable
3. ระบบส่งอีเมล/ข้อความพร้อมไฟล์อัตโนมัติ
4. ติด tag ตามประเภทลูกค้า เช่น แม่ค้าออนไลน์ / ร้านอาหาร / สนใจ audit
5. ส่ง follow-up sequence 2–3 ข้อความ
6. แจ้งเตือนเจ้าของเพจเมื่อมี lead ร้อน

## เวอร์ชัน 3: Full funnel ที่ควรไปถึงภายหลัง
- Landing page สำหรับ freebie
- Checkout page สำหรับ Prompt Pack
- ระบบส่งไฟล์อัตโนมัติหลังชำระเงิน
- CRM lead status
- Dashboard วัด conversion

## Data fields ที่ควรเก็บ
- วันที่เข้ามา
- ชื่อ
- Facebook profile / ชื่อเพจ
- ประเภทธุรกิจ
- pain point หลัก
- สนใจ freebie อะไร
- ซื้อ pack หรือยัง
- สนใจ audit หรือไม่
- สถานะล่าสุด

## Message sequence แนะนำ
### หลังส่งฟรีทันที
ส่งไฟล์ให้ + ถามธุรกิจของเขา

### Follow-up 1 (ภายใน 1 วัน)
ถามว่าได้ลองใช้หรือยัง และติดตรงไหน

### Follow-up 2 (ภายใน 2–3 วัน)
เสนอ Prompt Pack ที่ตรงกลุ่ม

### Follow-up 3
ถ้าดูจริงจัง เสนอ audit

## KPI ที่ควรวัด
- จำนวนคนคอมเมนต์ keyword
- จำนวน DM ที่ตอบกลับ
- จำนวนคนโหลดไฟล์ฟรี
- จำนวนคนซื้อ pack
- จำนวนคนที่นัดคุยต่อ
- conversion rate แต่ละขั้น

## สิ่งที่ควรทำต่อเชิงระบบ
1. ทำ Google Sheet template สำหรับเก็บ lead
2. ทำลิงก์ Drive สำหรับ freebie และ pack
3. ทำข้อความตอบกลับแยกตามประเภทลูกค้า
4. ทำฟอร์มสั่งซื้อแบบง่าย
5. ทำขั้นตอนส่งไฟล์หลังรับเงิน

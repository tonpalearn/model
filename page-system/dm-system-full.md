# DM System - Full Flow

## 1) ตอบคนที่คอมเมนต์ขอไฟล์ฟรี
สวัสดีครับ/ค่ะ ขอบคุณที่สนใจชุด Prompt ฟรีของต้นทาง AI 😊
นี่คือไฟล์สำหรับเริ่มใช้งาน:
[วางลิงก์ไฟล์]

ถ้าสะดวก ขอถามสั้นๆ 1 ข้อครับ/ค่ะ
ตอนนี้คุณใช้ AI กับงานแบบไหนมากที่สุด?
- ขายของออนไลน์
- ทำคอนเทนต์
- งานออฟฟิศ
- ยังไม่เคยใช้

## 2) ถ้าตอบว่าขายของออนไลน์
ถ้าเน้นงานขายของ ตอนนี้มี Prompt Pack ที่จัดไว้ให้ใช้ง่ายมากครับ/ค่ะ
ใช้กับงานพวกนี้ได้เลย:
- เขียนโพสต์ขาย
- ตอบลูกค้า
- คิดโปรโมชัน
- วางคอนเทนต์

ถ้าสนใจ เดี๋ยวส่งรายละเอียดชุดเริ่มต้นให้ดูได้ครับ/ค่ะ

## 3) ส่งรายละเอียด Prompt Pack
ตอนนี้มี **Prompt Pack แม่ค้าออนไลน์** ราคา 149 บาทครับ/ค่ะ
เหมาะกับคนที่อยากใช้ AI ให้ช่วยขายแบบง่ายๆ โดยไม่ต้องคิด prompt เองตั้งแต่ศูนย์

ในชุดมี prompt สำหรับ:
- เขียนโพสต์ขาย
- ตอบลูกค้าถามราคา
- ตอบลูกค้าที่ยังลังเล
- คิดโปรโมชัน
- วางคอนเทนต์
- สคริปต์ขายผ่านแชต

ถ้าสนใจ เดี๋ยวส่งวิธีรับไฟล์ให้ได้เลยครับ/ค่ะ

## 4) ถ้าลูกค้าบอกขอดูก่อน
ได้เลยครับ/ค่ะ
ถ้าอยากลองก่อน แนะนำเริ่มจากชุดฟรีที่ส่งให้ แล้วถ้ารู้สึกว่าใช้งานโอเค ค่อยขยับมาชุดเต็มก็ได้

ถ้ามีตัวอย่างสินค้าหรือโพสต์ที่อยากลองปรับ ส่งมาได้ เดี๋ยวช่วยดูแนวทางให้ครับ/ค่ะ

## 5) ถ้าลูกค้าสนใจซื้อ
ได้เลยครับ/ค่ะ 🙌
Prompt Pack แม่ค้าออนไลน์ ราคา 149 บาท
ถ้าพร้อม เดี๋ยวส่งรายละเอียดการชำระเงินและวิธีรับไฟล์ให้ได้ครับ/ค่ะ

## 6) หลังส่งไฟล์แล้ว
ขอบคุณมากครับ/ค่ะ 🙏
นี่คือไฟล์ของคุณ:
[วางลิงก์]

ถ้าลองใช้แล้วติดเรื่องการปรับ prompt ให้เข้ากับสินค้า ส่งตัวอย่างมาได้ เดี๋ยวช่วยแนะนำให้ครับ/ค่ะ

## 7) Follow-up หลัง 1 วัน
สวัสดีครับ/ค่ะ ขอเช็กสั้นๆ ว่าได้ลองใช้ไฟล์ที่ส่งไปหรือยัง
ถ้ามีจุดไหนอยากให้ช่วยแนะนำเพิ่มเติม ส่งตัวอย่างมาได้เลยครับ/ค่ะ

## 8) Follow-up เพื่อ upsell audit
ถ้าคุณอยากรู้ว่า AI ควรเริ่มใช้ตรงไหนก่อนในงานของตัวเอง ตอนนี้มี **AI Starter Audit** สำหรับมือใหม่ครับ/ค่ะ
เหมาะกับคนที่อยากเริ่มแบบมีทิศทาง และไม่ต้องลองผิดลองถูกเองทั้งหมด

## 9) ตอบคนที่สนใจ Audit
AI Starter Audit เหมาะกับเจ้าของกิจการเล็กหรือคนที่อยากใช้ AI กับงานจริงครับ/ค่ะ
สิ่งที่จะได้ เช่น:
- ดูว่างานไหนควรใช้ AI ก่อน
- แนะนำเครื่องมือพื้นฐาน
- แนะนำ prompt ให้เหมาะกับงานของคุณ

ถ้าสนใจ เดี๋ยวส่งรายละเอียดให้ต่อได้ครับ/ค่ะ

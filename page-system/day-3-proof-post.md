# Day 3 Proof Post

คำสั่งแบบเดิม:
“ช่วยเขียนโพสต์ขายกระเป๋า”

ผลลัพธ์ที่ได้มักจะกว้าง และไม่ค่อยจับใจลูกค้า

แต่ถ้าปรับ prompt ให้ชัดขึ้น เช่น:
“ช่วยเขียนโพสต์ขายกระเป๋าสำหรับผู้หญิงวัยทำงาน ที่อยากได้ลุคเรียบแต่ดูแพง จุดเด่นคือเบา จุของได้ และแมตช์ง่าย โทนภาษากันเองแต่ดูน่าเชื่อถือ”

ผลลัพธ์ที่ได้จะใช้งานจริงง่ายกว่าเยอะ

นี่คือเหตุผลว่าทำไม prompt ที่ดี ช่วยประหยัดเวลาได้มาก

ถ้าอยากได้ชุด prompt พร้อมใช้สำหรับขายของออนไลน์
พิมพ์คำว่า **PROMPT** ได้เลย

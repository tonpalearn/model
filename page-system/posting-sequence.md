# Posting Sequence - ต้นทาง AI

## ลำดับโพสต์แนะนำ 7 วันแรก

### Day 1
โพสต์เปิดตัวเพจ
เป้าหมาย: บอกว่าเพจนี้ช่วยใคร และให้คนเริ่มคอมเมนต์รับของฟรี

### Day 2
โพสต์แจก 20 Prompt ฟรี
เป้าหมาย: เก็บ lead

### Day 3
โพสต์ Before/After การใช้ AI ช่วยเขียนโพสต์ขาย
เป้าหมาย: สร้างความเชื่อว่าของนี้ใช้ได้จริง

### Day 4
โพสต์ “3 ความผิดพลาดที่ทำให้ใช้ AI แล้วไม่เวิร์ก”
เป้าหมาย: ให้ความรู้ + ปูทางสินค้า

### Day 5
โพสต์ขาย Prompt Pack แม่ค้าออนไลน์
เป้าหมาย: conversion ตัวแรก

### Day 6
โพสต์ตัวอย่าง Prompt Pack ร้านอาหาร/คาเฟ่
เป้าหมาย: ทดสอบกลุ่มร้านอาหาร

### Day 7
โพสต์ขาย AI Starter Audit
เป้าหมาย: ดึงคนที่อยากได้คำแนะนำเฉพาะตัว

## หลักคิด
- อย่าขายทุกโพสต์
- สลับระหว่างแจกของ / สอน / proof / ขาย
- ใช้ keyword ให้ชัด เช่น PROMPT / PACK / AUDIT

# PDF Export Guide

ไฟล์ที่พร้อมเอาไปจัดเป็น PDF
- `lead-magnet-20-prompts-sales-ready.md`
- `prompt-pack-แม่ค้าออนไลน์-sales-ready.md`

## วิธีเร็วที่สุด
1. เปิดไฟล์ markdown
2. คัดลอกไปวางใน Google Docs / Word / Canva Docs
3. ใส่โลโก้เพจ + สีแบรนด์
4. export เป็น PDF

## โครงสร้างหน้าที่แนะนำ
### Lead Magnet ฟรี
- หน้าปก: ชื่อไฟล์ + ชื่อเพจ
- หน้า 1: คำนำ + วิธีใช้
- หน้า 2-4: 20 prompts
- หน้าสุดท้าย: CTA ไป Prompt Pack

### Prompt Pack ขายจริง
- หน้าปก
- เหมาะกับใคร
- วิธีใช้
- prompts หลัก
- tips ใช้งาน
- หน้าสุดท้าย: ติดต่อเพจ / upsell audit

## โทนงานแนะนำ
- เรียบ สะอาด อ่านง่าย
- ใช้สีไม่เกิน 2–3 สี
- เน้นความน่าเชื่อถือ ไม่ต้อง flashy เกินไป

# Graphic Brief — Facebook Wow Post "Prompt Drop"

โปรเจกต์: ต้นทาง AI  
วัตถุประสงค์: ทำโพสต์แบบ wow สำหรับวันนี้ ที่ “แจก prompt ใช้งานจริง” ในโพสต์เลย เพื่อดึง reach / save / share / comment และตอก positioning ว่าเพจนี้ให้ของที่ใช้ได้จริง ไม่ใช่แค่พูดเรื่อง AI ลอยๆ  
กลุ่มเป้าหมาย: มือใหม่เรื่อง AI / เจ้าของธุรกิจเล็ก / แม่ค้าออนไลน์ / คนขายของใน Facebook, LINE, IG  
โทนแบรนด์: ไทย-first, เรียบ สะอาด ดูน่าเชื่อถือ, ไม่เทคเกินไป, อ่านจบไวบนมือถือ

---

## 1) Recommendation: Format ที่ควรใช้

### แนะนำ: Carousel 5 slides

**เหตุผล**
- โพสต์นี้ต้องการ “แจก prompt จริง” หลายตัวในโพสต์เดียว
- Carousel ทำให้คนปัดดูต่อ, ใช้พื้นที่ได้มากกว่า single image
- แต่ละสไลด์ใส่ prompt ได้ชัด อ่านง่ายกว่าบีบลงภาพเดียว
- เหมาะกับ Facebook organic และสามารถนำไป boost ads ได้
- เพิ่มโอกาส save/share เพราะคนรู้สึกว่าเป็น “ของเอาไปใช้ได้เลย”

### ไม่แนะนำเป็น single image สำหรับชิ้นนี้
- ถ้าใส่ prompt หลายข้อในภาพเดียว จะตัวเล็กเกินไปบนมือถือ
- ถ้าลดเหลือ prompt เดียว ความ wow จะหาย

---

## 2) Big Idea ของชิ้นงาน

### Concept
**แจก 3 Prompt ทำเงิน ที่มือใหม่เอาไปวางใน ChatGPT แล้วใช้ต่อได้ทันที**

### Message หลัก
- AI ไม่ต้องเริ่มจากการเรียนลึก
- เริ่มจาก “prompt ที่ใช้งานได้จริง” ก่อน
- ต้นทาง AI = เพจที่คัดของให้แล้วสำหรับคนทำมาหากิน

### Desired Reaction
- “อันนี้เอาไปใช้ได้จริง”
- “เซฟไว้ก่อน”
- “เพจนี้ไม่ได้พูดลอยๆ มีของให้หยิบไปใช้”

## 3) Headline / Subheadline Options

### Headline options
1. แจก 3 Prompt ทำเงิน
2. 3 Prompt ที่คนขายของควรเซฟ
3. เอา Prompt ชุดนี้ไปใช้ได้เลย
4. มือใหม่ก็ใช้ AI ช่วยขายได้
5. Prompt พร้อมใช้ สำหรับคนขายของออนไลน์

### Subheadline options
1. คัดมาให้แล้ว สำหรับมือใหม่ที่อยากใช้ AI แบบได้งานจริง
2. แค่คัดลอกไปใส่ ChatGPT แล้วปรับชื่อสินค้า
3. ช่วยคิดโพสต์ ช่วยตอบลูกค้า ช่วยคิดโปรในไม่กี่นาที
4. ไม่ต้องเก่งเทค ก็เริ่มใช้ AI กับงานขายได้ทันที

**Headline:** แจก 3 Prompt ทำเงิน

## 5) Layout Guidance

### Overall layout system
- ใช้กริดเรียบ 1 คอลัมน์ เป็นหลัก
- ทุกสไลด์เน้นข้อความกลางภาพหรือชิดซ้ายแบบอ่านเร็ว
- มีพื้นที่หายใจเยอะ ไม่อัดข้อความชิดขอบ
- ใส่กรอบหรือ card สำหรับส่วน prompt เพื่อให้คนมองรู้ทันทีว่า “นี่คือของเอาไปใช้ได้”

### Recommended composition per slide
- ด้านบน: Label เล็ก / category เช่น PROMPT 1
- กลางบน: Headline ใหญ่
- กลาง: Prompt box หรือข้อความหลักในกล่องพื้นตัดสี
- ด้านล่าง: tip / CTA สั้น
- มุมล่าง: ชื่อเพจ ต้นทาง AI

### Mobile-first spacing
- margin รอบงานค่อนข้างกว้าง
- บรรทัดต่อย่อหน้าไม่ยาวเกินไป
- หลีกเลี่ยงข้อความหลายระดับเกินจำเป็น
- ถ้าข้อความยาว ให้แบ่งบรรทัดจงใจ ไม่ปล่อยเป็นบล็อกยาว

## 6) Visual Hierarchy

### ลำดับสายตาที่ต้องการ
1. Headline ใหญ่ที่สุด
2. Prompt box หรือเนื้อหาหลัก
3. Subheadline / tip
4. CTA
5. Footer brand

### ขนาดโดยประมาณ
- Headline: ใหญ่ หนา ชัดมาก
- Prompt text: กลางค่อนไปเล็ก แต่ยังอ่านได้จริงบนมือถือ
- Label เช่น PROMPT 1: เล็กแต่เด่นด้วยสี accent
- Footer: เล็กสุด

### Emphasis rules
- ไฮไลต์คำสำคัญ เช่น “โพสต์ขาย”, “ตอบลูกค้า”, “คิดโปร”
- ใช้ bold เฉพาะหัวสำคัญ อย่า bold ทั้งย่อหน้า
- อย่าใช้หลายสีแข่งกันในข้อความเดียว

---

## 7) Color / Style Guidance

ยึดจากไฟล์แบรนด์เดิม: สะอาด มินิมอล น่าเชื่อถือ และไม่เทคเกินไป

### Recommended palette
- Background หลัก: ขาว หรือ off-white
- Primary text: ดำ / เทาเข้ม
- Accent: ฟ้าน้ำเงิน หรือฟ้าอ่อน
- Optional support accent: เทาอ่อนสำหรับกล่อง / divider

### Style cues
- clean knowledge brand
- minimal but not sterile
- ดูเหมือน “ชีทสรุปที่มีมูลค่า”
- เน้นความเป็นเครื่องมือใช้งานจริง มากกว่างานกราฟิกหวือหวา

### Avoid
- ไล่สีแรงๆ เยอะเกินไป
- ไอคอนรก
- AI futuristic neon style
- stock photo ที่ดู generic
- ข้อความแน่นเต็มภาพ

---

## 8) Typography Guidance

### ฟอนต์
- ใช้ฟอนต์ไทยอ่านง่าย เช่นแนว modern sans ไทย
- หัวใช้ weight หนา
- เนื้อหา prompt ใช้ regular/medium ที่ยังชัดบนมือถือ

### Typography behavior
- หัวต้องอ่านจบไวใน 2 วินาที
- Prompt ในกล่องควรจัดบรรทัดให้ดู “copy ง่าย”
- ระยะบรรทัดค่อนข้างโปร่ง
- ไม่ใช้ตัวบางมาก

---

## 9) Thumbnail-Safe Text

บนหน้า feed คนอาจเห็นแค่สไลด์แรก/thumbnail เป็นหลัก  
ดังนั้นสไลด์แรกต้องชัดแม้ยังไม่ได้กดเข้าไปอ่าน

### ข้อความ thumbnail-safe ที่แนะนำ
**แจก 3 Prompt ทำเงิน**

### Alternate thumbnail-safe options
- 3 Prompt ที่ควรเซฟ
- Prompt พร้อมใช้สำหรับคนขายของ
- มือใหม่ก็ใช้ AI ช่วยขายได้

### Rule
- สไลด์แรกอย่าใส่ข้อความเกิน 3–4 บล็อก
- Headline ต้องกินพื้นที่เด่นชัดที่สุด

## 10) CTA Placement

### In-graphic CTA
- Slide 1: “ปัดดูแล้วเซฟไว้”
- Slide 5: “คอมเมนต์ PROMPT” + “กดเซฟ • กดแชร์”

- ชวนเซฟโพสต์ไว้ใช้วันคิดโพสต์ไม่ออก
- ถ้ามีทีมตอบคอมเมนต์พร้อม ให้ใช้ CTA แบบ comment keyword เป็นหลัก

## 11) Suggested Facebook Caption

ขายของออนไลน์ แต่คิดโพสต์ไม่ออก ตอบลูกค้าไม่ทัน หรือคิดโปรไม่ออกบ่อยไหม

นี่คือ 3 prompt ที่เอาไปวางใน ChatGPT แล้วใช้ต่อได้ทันที
เหมาะมากสำหรับมือใหม่ที่อยากเริ่มใช้ AI แบบ “ได้งานจริง” ไม่ใช่แค่อ่านแล้วจบ

ในโพสต์นี้มี prompt สำหรับ:
- เขียนโพสต์ขาย
- ตอบลูกค้าถามราคา
- คิดโปรโมชัน

วิธีใช้สั้นๆ:
1. คัดลอก prompt
2. ใส่ชื่อสินค้า/กลุ่มลูกค้า/รายละเอียดร้านของตัวเอง
3. วางใน ChatGPT
4. ปรับภาษาอีกนิดก่อนโพสต์จริง

ถ้าอยากให้ต้นทาง AI แจก prompt ชุดต่อไป
คอมเมนต์คำว่า **PROMPT** ไว้ได้เลย

และถ้าเห็นว่ามีประโยชน์ เซฟโพสต์นี้ไว้ใช้วันคิดไม่ออกได้เลย

#ต้นทางAI #AIสำหรับมือใหม่ #Promptพร้อมใช้ #แม่ค้าออนไลน์ #เจ้าของธุรกิจเล็ก

## 13) Export Specs

### Facebook Organic
- Size แนะนำ: 1080 x 1350 px
- Ratio: 4:5
- Format: PNG
- จำนวน: 5 slides
- Safe margin: เว้นขอบทุกด้านอย่างน้อย 80–100 px

### Facebook Ads Compatible
- ใช้ขนาดเดียวกันได้: 1080 x 1350 px
- ตรวจให้ข้อความสำคัญไม่ชิดขอบ
- ไฟล์ต้องคม อ่านง่ายบนมือถือ
- หลีกเลี่ยงข้อความแน่นเกินไปเพื่อคง performance

### Cover / Thumbnail priority
- Slide 1 ต้องอ่านรู้เรื่องแม้ย่อเล็กใน feed
- หัวใหญ่ต้องชัดกว่าส่วนอื่นมาก

## 14) Production Notes for Designer / Operator

- ถ้ามีเวลาทำแค่เวอร์ชันเดียว ให้ทำตาม brief นี้ก่อน
- ถ้าจะทำ A/B ภายหลัง ให้เปลี่ยนเฉพาะ Slide 1 headline ได้เลย
- ถ้าอยากเพิ่มแรงเซฟ/share อาจใส่ badge เล็กบนปกว่า “เซฟไว้ใช้ได้จริง”
- อย่าเพิ่มข้อความอธิบายยาวเกินจาก brief เพราะจะทำให้ภาพแน่น

## 15) Recommended Final Direction

ถ้าต้องเลือกเวอร์ชันเดียวสำหรับโพสต์วันนี้:

**ใช้ Carousel 5 slides**

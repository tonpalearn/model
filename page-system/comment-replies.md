# Comment Replies

## คนพิมพ์ PROMPT
ขอบคุณครับ/ค่ะ เดี๋ยวส่งรายละเอียดให้ทางข้อความนะ 😊

## คนพิมพ์ PACK
รับทราบครับ/ค่ะ เดี๋ยวส่งรายละเอียดชุด Prompt Pack ให้ทางข้อความนะ

## คนพิมพ์ AUDIT
ได้เลยครับ/ค่ะ เดี๋ยวส่งรายละเอียด AI Starter Audit ให้ทางข้อความนะ

## คนถามว่าใช้ฟรีไหม
มีชุดฟรีให้เริ่มก่อนครับ/ค่ะ ถ้าอยากได้ พิมพ์คำว่า PROMPT ได้เลย

## คนถามว่าใช้กับอะไรได้บ้าง
ใช้กับงานขาย งานคอนเทนต์ และงานสื่อสารกับลูกค้าได้ดีมากครับ/ค่ะ เดี๋ยวถ้าสนใจเริ่มต้น ส่งชุดฟรีให้ลองก่อนได้

## คนถามว่าใช้กับ ChatGPT ได้ไหม
ได้ครับ/ค่ะ ชุดที่ทำไว้ตั้งใจให้ใช้กับ ChatGPT ได้ทันที

## คนถามราคา
ตอนนี้ชุดเริ่มต้นราคา 149 บาทครับ/ค่ะ ถ้าสนใจ เดี๋ยวส่งรายละเอียดให้ทางข้อความได้

## คนถามว่ามีของสายร้านอาหารไหม
มีครับ/ค่ะ ตอนนี้มี Prompt Pack ร้านอาหาร/คาเฟ่ ด้วย ถ้าสนใจ เดี๋ยวส่งรายละเอียดให้ทางข้อความได้

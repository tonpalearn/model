# Lead Funnel - ต้นทาง AI

## Funnel เป้าหมาย
โพสต์ความรู้/แจกของฟรี -> คอมเมนต์ keyword -> DM ส่ง freebie -> เสนอ Prompt Pack ราคาง่าย -> นัดคุย audit/advisory

## Step-by-step
1. โพสต์พร้อม hook ชัดเจน
2. ให้คนคอมเมนต์ keyword เช่น PROMPT / ร้านอาหาร / แม่ค้า
3. ส่ง DM พร้อม freebie
4. ถามต่อ 1 คำถามเพื่อแยกกลุ่มลูกค้า
5. เสนอสินค้าที่ตรง pain
6. ถ้าสนใจลึกขึ้น เสนอ audit / advisory

## คำถามคัดกรองใน DM
- ตอนนี้ใช้ AI กับงานอะไรอยู่บ้าง?
- ขายสินค้าหรือบริการประเภทไหน?
- ปัญหาหลักคือคิดโพสต์ไม่ออก / ตอบลูกค้า / วางคอนเทนต์ / เริ่มใช้ AI ไม่เป็น?

## KPI เริ่มต้น
- จำนวนคอมเมนต์ต่อโพสต์
- จำนวน DM ที่ตอบกลับจริง
- จำนวนคนโหลด freebie
- conversion จาก freebie -> ซื้อ pack
- conversion จาก pack -> นัดคุย

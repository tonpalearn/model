# ต้นทาง AI - Page System

ระบบงานสำหรับเพจ Facebook "ต้นทาง AI" เพื่อขาย Prompt Pack และบริการแนะนำการใช้ AI เบื้องต้น

## เป้าหมาย
- สร้างคอนเทนต์ที่ดึงคนสนใจ AI มือใหม่/เจ้าของธุรกิจเล็ก
- เก็บ lead ผ่านคอมเมนต์/DM
- ปิดการขาย Prompt Pack ราคาง่าย
- upsell ไปยังบริการ audit / advisory

## ไฟล์สำคัญ
- `strategy.md` — กลยุทธ์แบรนด์และการขาย
- `offers.md` — สินค้า/บริการที่ควรขายก่อน
- `content-calendar-30d.md` — แผนโพสต์ 30 วัน
- `lead-funnel.md` — ขั้นตอนแปลงคนดูเป็นลูกค้า
- `dm-scripts.md` — สคริปต์ DM / คอมเมนต์ / ปิดการขาย
- `launch-checklist.md` — รายการทำงานให้ launch ได้จริง

## ข้อจำกัดปัจจุบัน
ยังไม่ได้โพสต์หรือส่งข้อความออกไปยัง Facebook อัตโนมัติ เพราะเป็นการกระทำภายนอก ต้องได้รับการอนุมัติจากผู้ใช้ก่อน

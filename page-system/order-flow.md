# Order Flow - Prompt Pack

## วิธีขายแบบง่ายที่สุดช่วงเริ่มต้น
1. ลูกค้าสนใจจากโพสต์หรือ DM
2. ส่งรายละเอียดสินค้า + ราคา
3. ลูกค้าโอนเงิน
4. ส่งไฟล์ผ่าน Drive / DM
5. บันทึกว่าเป็นลูกค้าแล้ว
6. Follow-up ขอ feedback ภายหลัง

## ข้อความส่งรายละเอียดสินค้า
ตอนนี้มี Prompt Pack แม่ค้าออนไลน์ ราคา 149 บาทครับ/ค่ะ
ในชุดจะมี prompt สำหรับ:
- เขียนโพสต์ขาย
- ตอบลูกค้า
- คิดโปรโมชัน
- วางคอนเทนต์
- ใช้กับ ChatGPT ได้ทันที

ถ้าสนใจ เดี๋ยวส่งรายละเอียดการชำระเงิน/วิธีรับไฟล์ให้ได้เลย

## ข้อความหลังลูกค้าจ่ายเงิน
ขอบคุณมากครับ/ค่ะ 🙏
นี่คือไฟล์ Prompt Pack ของคุณ:
[วางลิงก์]

ถ้าลองใช้แล้วติดตรงไหน ส่งตัวอย่างมาได้ เดี๋ยวช่วยแนะนำวิธีใช้ต่อให้ครับ/ค่ะ

## ข้อความ follow-up หลังส่งไฟล์ 2 วัน
สวัสดีครับ/ค่ะ อยากเช็กสั้นๆ ว่าได้ลองใช้ Prompt Pack แล้วหรือยัง
ถ้าติดเรื่องการปรับ prompt ให้เข้ากับสินค้าของคุณ ส่งตัวอย่างมาได้ เดี๋ยวช่วยดูให้ครับ/ค่ะ
